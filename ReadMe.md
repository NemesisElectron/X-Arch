> [!IMPORTANT]
> This project is still WIP (Working In Progress) and some bugs or issues could happen.\
> At this moment the project is working flawless on systems with AMD graphics cards and Intel processors.\
> For other graphics cards and processors, needs testers.

---

> [!WARNING]
> Requirements:
>> [KDEP](https://en.wikipedia.org/wiki/KDE_Plasma) as desktop environment,\
>> [SDB](https://en.wikipedia.org/wiki/Systemd-boot) as boot loader,\
>> [UEFI](https://en.wikipedia.org/wiki/UEFI) as firmware interface,\
>> [XFS](https://en.wikipedia.org/wiki/XFS) as file system,\
>> [GPT](https://en.wikipedia.org/wiki/GUID_Partition_Table) as partition table.

> [!CAUTION]
> Instructions:
>> Run on fresh CachyOS for no issues.
>>> ```fish
>>> sudo pacman --sync --noconfirm --needed dash git
>>> git clone --branch main --recurse-submodules https://github.com/NemesisElectron/X-Arch.git /tmp/X-Arch
>>> dash /tmp/X-Arch/Setup.sh
>>> ```
